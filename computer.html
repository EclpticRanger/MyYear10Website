Binary & Hexadecimal

Binary is a base-2 numeral system that uses only two digits, 0 and 1 this is used to write data in computers. Hexadecimal, or base-16, uses sixteen symbols: 0-9 and A-F. It is often used in computing as a more human-friendly representation of binary-coded values.

ASCII

ASCII (American Standard Code for Information Interchange) is a character encoding standard that uses numbers to represent characters. Each letter, digit, and symbol is assigned a unique number from 0 to 127. For example, the letter 'A' is represented by the number 65 in ASCII.

Error Detection

Error detection is a technique used to identify errors in data transmission or storage. Common methods include parity bits, checksums, and cyclic redundancy checks (CRC). These methods add extra bits to the data to help detect errors that may occur during transmission.
